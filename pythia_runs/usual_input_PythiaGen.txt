# A faster compilation test -- Using O3's loop unrolling, inlining and operation vectorization (SIMD):
    # This could be better for my code, as I explicitly optimized it to avoid cache misses and stuff like that!

# Additional compiling optimizations:
1) -funroll-loops
Unrolls some inner loops!
2) -ffp-contract=fast
Contracts expressions like (a * b + c) into a single processor instruction
3) -fuse-linker-plugin
More information for the -flto optimization!
4) -fopenmp-simd
Allows for OpenMP to use SIMD instructions too.
# Did not allow for the usage of -fno-math-errno, as it is not safe for Angantyr.

# Including these optimizations:
g++ -O3 -std=c++17 -march=native -funroll-loops -ffp-contract=fast -flto=2 -fuse-linker-plugin -pipe -fopenmp -fopenmp-simd PythiaGenMin.cxx -o PythiaGenMin.exe -I$PYTHIA8/include -L$PYTHIA8/lib -lpythia8 -I$FASTJET/include -L$FASTJET/lib -lfastjet `root-config --cflags --libs`
