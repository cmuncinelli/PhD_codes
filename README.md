General purpose repository for my PhD.

- [**vorticity_hydro**](vorticity_hydro): Contains a few hydro-based simulations for proof-of-concept codes.
- [**pythia_runs**](pythia_runs): Pythia generator codes. Mostly complimentary to the vorticity_hydro study as a way to generate data that should have zero signal. Both for PYTHIA8 and PYTHIA Angantyr.
